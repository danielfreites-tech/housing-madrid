# housing-madrid

Web app that predicts housing purchase prices in Madrid using machine learning.

## Stack

| Layer    | Technology              |
|----------|-------------------------|
| Frontend | React                   |
| Backend  | FastAPI (Python)        |
| Model    | CatBoost                |
| Dataset  | houses_Madrid.csv       |

## Structure

```
├── data/          # Dataset (not tracked)
├── model/         # Notebooks, training utils, artifacts
├── backend/       # FastAPI app
└── frontend/      # React app
```
